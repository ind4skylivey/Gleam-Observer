# Assets Directory

Place the GleamObserver icon here:

- **gleam-observer.png** (256x256 px recommended)
- **gleam-observer.svg** (vector format, preferred for scalability)

The install script will automatically detect and install the icon.

## Icon Requirements
- Format: PNG or SVG
- Size: 256x256 px (PNG) or scalable (SVG)
- Style: Should represent system monitoring / hardware
- Colors: Catppuccin palette recommended
